# Icon Gallery Licenses

The built-in image gallery bundles normalized offline icon data from
[Iconify](https://iconify.design/). Iconify provides the common JSON format; each icon
set remains licensed by its original authors.

| Icon set | Author | License | Source |
| --- | --- | --- | --- |
| Material Design Icons | Pictogrammers | Apache-2.0 | <https://github.com/Templarian/MaterialDesign> |
| Carbon | IBM | Apache-2.0 | <https://github.com/carbon-design-system/carbon> |
| Bootstrap Icons | The Bootstrap Authors | MIT | <https://github.com/twbs/icons> |

The Apache License 2.0 text is included in the repository root `LICENSE`. The MIT and
MIT notice distributed by the upstream project applies to Bootstrap Icons. Icons
selected from any gallery set are converted to standalone SVG data URIs and embedded in
the diagram.
